AGOS — logomark options

Six directions for the AGOS wordmark + glyph. Each shown in dark nav, light hero, at favicon / nav / hero sizes, plus monochrome fallbacks. The current site uses a flat "A" in a blue‑to‑navy square — a strong starting point but generic; every option below keeps that 26×26 footprint and blue/navy palette.

Constraints. Must read at 16px (favicon) and 26px (nav). Dark and light backgrounds. Default accent #2f6feb, navy #0b1a3a. Wordmark set in Geist Medium, tight tracking. Meaning to evoke: a digital COO orchestrating twelve agents — systematic, compounding, auditable.
